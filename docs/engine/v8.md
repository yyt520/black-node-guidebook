V8 是 Google 机遇 C++ 编写的开源高性能 JavaScript 与 WebAssembly 引擎，主要应用包括 Chrome 浏览器以及 Node.js。得益于 Chrome 浏览器的市场占有率以及 Chromium 阵营的不断强大，V8 已经成为了当今最主流的 JavaScript 引擎。
