# 安全设计

常见的安全漏洞：

- XSS 攻击：对 Web 页面注入脚本，使用 JavaScript 窃取用户信息，诱导用户操作
- CSRF 攻击：伪造用户请求向网站发起恶意请求
- 钓鱼攻击：利用网站的跳转链接或者图片制造钓鱼陷阱
- HTTP 参数污染：利用对参数格式验证的不完善，对服务器进行参数注入攻击
- 远程代码执行：用户通过浏览器提交执行命令，由于服务端没有针对执行函数做过滤，导致没有指定绝对路径的情况下就执行命令
